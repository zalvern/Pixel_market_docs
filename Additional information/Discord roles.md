# Discord role level system:
The system aims to provide a more effortless way to manage discord perms, by instead of assigning permission to each placement role, a clearance levels system is used.

Textual roles are purely cosmetic, and only L-X roles give perms.

L-1: Given to everyone, who joins the server via the dyno bot. This role allows you to chat and do basic interactions in the server.
L-2: Given to all members of the business (AKA-Staff), This role lets its bearer use the server's structure and tools, to do their assigned job. Like being able to close threads in the sales forums.
L-3: Gives basic server management action perms, like the ability to kick, mute or ban people. Given to moderators, and higher ranking staff members.
L-4: Given to High Command, this level gives almost full access to all settings and stuff, it is for people, who run things more than actually doing staff stuff.
L-5: Absolute control, only given to the directorate. Gives admin.
